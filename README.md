# 🎣 HobbyHook

> Discover meaningful hobbies and reclaim your screen time with personalized activity recommendations.

## 🌟 About the Project

HobbyHook is a React-based web application that helps users break free from endless scrolling by recommending engaging activities based on their interests and available time.

Whether you enjoy creative activities, physical exercises, analytical challenges, or social interactions, HobbyHook suggests personalized hobbies to help you spend your time in a more fulfilling way.

---

## ✨ Features

* 🎨 Choose from multiple hobby categories:

  * Creative
  * Physical
  * Analytical
  * Social

* ⏰ Select the amount of free time you have:

  * 10 minutes
  * 30 minutes
  * 60 minutes

* 🎲 Randomized hobby recommendations.

* 🔄 Prevents showing the same recommendation twice in a row.

* 📱 Responsive and colorful user interface.

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/Parv58/HobbyHook.git
```

### Navigate to the project

```bash
cd HobbyHook
```

### Install dependencies

```bash
npm install
```

### Run the development server

```bash
npm start
```

The application will be available at:

```text
http://localhost:3000
```
